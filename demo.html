Quick Start Guide

1. Encryption/Decryption:

  1. Select your preferred encryption method (AES, RSA, DES, etc.)
  2. Choose between Encrypt or Decrypt mode
  3. Enter your text and secret key
  4. Configure options (key size, mode, format)
  5. Click the action button to process
  6. Copy the result using the copy button

2. Hashing:

  1. Select a hashing algorithm (MD5, SHA256, Bcrypt, HMAC)
  2. Enter the text you want to hash
  3. For HMAC, provide a secret key
  4. Click to generate the hash

3. Encoding/Decoding:

  1. Select encoding format (Base64, Hex)
  2. Choose Encode or Decode operation
  3. Enter your text
  4. Process and copy the result

4. Calculators:

  1. Select the calculator type from the sidebar
  2. Fill in the required fields
  3. Click calculate to get results
  4. Results are displayed with detailed breakdown
